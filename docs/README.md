# MyMantra

**Application:** MyMantra — spiritual practice companion
**Stack:** Flutter (Dart), Riverpod, go_router, SharedPreferences

---

## About

MyMantra is a cross-platform offline-first application for mantra meditation. Users create and manage mantras, schedule reminders, track repetitions with a tap counter, and build daily practice streaks. Cloud sync and audio features are planned for Phase 2.0; gamification for Phase 3.0.

Full product vision and scope: [product/application_foundations.md](product/application_foundations.md)

---

## Getting Started

1. Install prerequisites and toolchain:
   ```
   make install
   ```
2. Run the app on a device or simulator:
   ```
   make debug
   ```
3. For build targets, `target.json` configuration, and per-platform constraints, see [software/build-system-architecture.md](software/build-system-architecture.md).
4. For the source code layout, see [software/folder_structure.md](software/folder_structure.md).
